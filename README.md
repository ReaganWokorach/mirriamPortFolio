## Design Features
- **Professional Palette:** A sophisticated Rose-Gold and Slate Grey theme.
- **Glassmorphism:** Modern blurred navigation bar for a premium feel.
- **Responsive Navigation:** A fully functional hamburger menu for mobile devices.
- **Animated Timeline:** A sleek visual representation of professional experience.
- **SEO Optimized:** Structured HTML5 for better search engine visibility.

## Built With
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling with Flexbox, Grid, and Media Queries.
- **JavaScript (ES6):** Interactive mobile menu logic.
- **Google Fonts:** 'Playfair Display' for headings and 'Plus Jakarta Sans' for body text.
- **Font Awesome:** High-quality vector icons.

## How to Setup Locally
1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```
2. **Navigate to the folder:**
   ```bash
   cd mirriam-portfolio
   ```
3. **Add the profile photo:**
   Ensure your professional photo is saved as `profile.jpg` in the root folder.
4. **Open in Browser:**
   Simply double-click `index.html` to view the site locally.
